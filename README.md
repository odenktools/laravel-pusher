# Up and Running

```bash
composer install
```

```bash
cp .env.example .env
```

```bash
php artisan key:generate
```

```bash
php artisan vendor:publish
```

```bash
composer dumpautoload
```

Open ```.env``` and add your ```PUSHER CREDENTIALS```

```
PUSHER_APP_ID=xxxxxxxxx
PUSHER_KEY=xxxxxxxxxxx
PUSHER_SECRET=xxxxxxxxxx
```
